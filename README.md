# REST-Assured
ETA_2022_2B -  Testes de Serviços  com REST Assured
Projeto de avaliação de automação de testes de serviços, da disciplina Testes de Serviços

# Requisitos

- Gradle 8.3
- IntelliJ IDEA
- Java 20
- JUnit
- Rest assured

#Cenários automatizados

- Cadastrar usuáio
- Editar usuário
- Listr usuário
- Listar usuário por ID
- Excluir usuário por ID
  
- Cadastrar produto
- Listar produto
- Excluir produto
  
- Cadastrar Carrinho
- Cancelar Compra
- Verificar Carrinho

#Tecnologias
- Liguagem Java
- RestAssured
- Rest APIs
